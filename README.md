# Algoharjoitus  

Ohjelma suorittaa Delaunayn triangulaation käyttäen Bowyer-Watsonin algoritmia.  
ESC = Sulje ohjelma.  
R = Uudelleengeneroi triangulaatio.  
G = Generoi huoneet.  

## Testaus  

Avaa virtuaaliympäristö:
```
poetry shell
```

Aja testit:  
```
pytest
```
